quickstart

The AppGeo/geo plugin has one widget, geomap, that you can instantiate on any div with a computable width and height.

$("#map").geomap();

By default, this one line of code will create a fully functional map on your page showing the whole world and using the OpenStreetMap tile set.

The default units are web mercator meters which is the same unit type that Google, Bing and ESRI now use. You can set the center and zoom of the map at the same time you initialize it by passing a JavaScript object of options.

$("#map").geomap({
  center: { x: -7912531.856997, y: 5214932.836549 },
  zoom: 10
});

The above example will show the City of Boston. If you need help determining the center point values the Spatial Reference .org website can help. Pan and zoom the map around until the marker is on the location you want and copy the Output Coordinates to you code.
